# Superstore Sales Performance Dashboard

## Overview
This project presents an interactive Tableau dashboard analyzing sales performance using the Sample Superstore dataset.  
The dashboard highlights key business metrics, sales trends over time, and geographic performance across U.S. regions.

- KPI tiles for Sales, Profit, and Profit Margin

- Sales by Region (Geographic Map)

## Insights
- Sales show strong growth in the second half of the year, peaking in November.
- Certain regions significantly outperform others, indicating opportunities for targeted regional strategies.
